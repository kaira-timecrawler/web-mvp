🎯 CAUSA PRINCIPAL
==================
La web muestra 'Site not found' porque GitHub Pages no está activado
en tu repositorio. A pesar de tener el workflow configurado, GitHub
no desplegará automáticamente a Pages hasta que lo actives manualmente.

🔧 SOLUCIÓN MANUAL (Recomendada)
=================================
Sigue estos pasos en la interfaz de GitHub:

1. Ve a tu repositorio:
   https://github.com/kaira-timecrawler/web-mvp

2. Ve a Settings > Pages
3. En 'Source', selecciona: 'Deploy from a branch'
4. Branch: 'main'
5. Folder: '/ (root)'
6. Haz clic en 'Save'

7. Espera 2-5 minutos para que GitHub procese el despliegue

🚀 SOLUCIÓN AUTOMATIZADA (Alternativa)
======================================
Si tienes acceso a GitHub CLI, puedes usar:

gh repo view kaira-timecrawler/web-mvp --json pages
gh pages create --source=main --branch=main --target=/
